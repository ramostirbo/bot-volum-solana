# Solana Volume, Sniper & Market Maker Bot 🚀

مشروع احترافي متكامل لإدارة وتداول العملات على شبكة سولانا (Solana). يجمع المشروع بين رفع حجم التداول، القنص السريع، وصناعة السوق.

---

## 📋 المتطلبات التشغيلية (Prerequisites)
1. تثبيت **Node.js** (إصدار 20 فما فوق) أو **Bun**.
2. محفظة سولانا (الأساسية) بها رصيد SOL كافٍ.
3. رابط **RPC Endpoint** (سولانا).
4. بوت تلغرام (للحصول على التنبيهات).

---

## ⚙️ الإعداد الأولي (Configuration)
قم بتعديل القيم داخل ملف `.env`:
*   `PRIVATE_KEY`: المفتاح الخاص لمحفظتك الأساسية.
*   `TOKEN_MINT`: عنوان العملة المستهدفة.
*   `RPC_ENDPOINT`: رابط الـ RPC.

---

## 🛠️ النظام الأول: رفع حجم التداول (Volume Maker)
يستخدم هذا النظام لخلق نشاط تداول مكثف وزيادة عدد حاملي العملة بشكل يبدو طبيعياً.

**طريقة التشغيل:**
```bash
bun index.js
```
**المميزات:**
*   يعرض اسم العملة وسعرها اللحظي.
*   ينشئ محافظ فرعية ويقوم بعمليات بيع وشراء عشوائية.

---

## 🎯 النظام الثاني: نظام القنص (Sniper Bot)
نظام فائق السرعة مصمم لشراء العملة فور إطلاقها في السوق أو توفر السيولة.

**خطوات العمل بالترتيب:**
1.  **إنشاء محافظ الهجوم:**
    ```bash
    node sniper-gen.js
    ```

2.  **تمويل المحافظ (الشحن):**
    ```bash
    node sniper-fund.js
    ```
    *يشحن المحافظ بمبلغ الـ SOL المحدد في الإعدادات.*

3.  **إطلاق الهجوم:**
    ```bash
    node sniper-bot.js
    ```
    *يراقب العملة ويطلق هجوماً متوازياً من جميع المحافظ فور توفر السعر.*

---

## 📈 النظام الثالث: صانع السوق (Market Maker)
نظام احترافي للحفاظ على استقرار السعر ودعم الشارت بعد الإطلاق.

**طريقة التشغيل:**
```bash
node market-maker.js
```
**الاستراتيجية:**
*   **Buy the Dip:** يشتري تلقائياً لدعم السعر عند الانخفاض.
*   **Profit Taking:** يبيع جزءاً بسيطاً عند الارتفاع لتوفير سيولة.
*   **مراقبة لحظية:** يعرض السعر ونسبة التغير كل 10 ثوانٍ.

---

## 🔍 أدوات إضافية (Tools)
*   **فحص السعر والاستجابة:**
    ```bash
    node check-price.js
    ```
    *يعرض اسم العملة، السعر الحالي، وسرعة استجابة الـ API (Latency).*

---

## 📝 ملاحظات هامة
*   **الأمان:** تم دمج نظام `cleanBase58` لتنظيف المدخلات من الأخطاء والمسافات.
*   **السرعة:** يستخدم البوت `DexScreener API` و `Jupiter API` لضمان أدق البيانات وأسرع تنفيذ.
*   **الخصوصية:** لا تشارك ملف `.env` أو `sniper_wallets.json` مع أي شخص.
